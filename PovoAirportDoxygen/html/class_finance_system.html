Detailed Description

Provides financial calculations for airport operations.

This class contains constants and utility methods used to calculate operational costs, revenues, and financial summaries over a specified period of time.

Calculates airport operating costs within a time period.

The calculation includes daily operating expenses as well as costs associated with arriving and departing flights recorded during the specified period.

Parameters
t1Start date and time of the period.
t2End date and time of the period.
Returns
float Total airport costs for the selected period.
